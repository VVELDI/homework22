# 🛒 Django Product Catalog + Blog

Простое, но функциональное Django-приложение для управления **каталогом продуктов** и ведения **блога** с полной административной и публичной частью.

---

## 🚀 Возможности

### 🛍 Каталог товаров
- Просмотр списка товаров
- Детальная страница каждого товара
- Управление через Django Admin
- Полная реализация на Class-Based Views (CBV)

### ✍️ Блог
- Создание, редактирование, удаление и просмотр блог-постов
- Заголовок, содержимое, изображение-превью, дата создания, флаг публикации, счётчик просмотров
- Фильтрация: отображаются только опубликованные записи
- Увеличение количества просмотров при заходе на страницу статьи
- Перенаправление на статью после редактирования
- Полностью реализован на CBV

### 📨 Контактная страница
- Форма обратной связи
- Обработка POST-запросов
- Сообщение об успехе / валидация данных

---

## ⚙️ Технологии

- Django 5.x
- PostgreSQL (через переменные окружения `.env`)
- Bootstrap 5 (вёрстка)
- CBV (Class-Based Views) по всему проекту
- Загрузка изображений через `MEDIA`
- Django Messages framework
