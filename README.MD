# Django Product Catalog

📦 Простое Django-приложение для управления каталогом продуктов и категорий.

## 🚀 Возможности

- CRUD-операции для продуктов и категорий
- Админ-панель Django
- Поддержка изображений (media)
- Работа с переменными окружения (.env)
- Кастомная management-команда для загрузки тестовых данных
- Фикстуры для моделей
- Удобный интерактивный Shell с ipython
